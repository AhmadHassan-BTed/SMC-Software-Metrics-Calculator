# Security Policy

## Supported Versions
Only the latest version is currently supported.

| Version | Supported |
| --- | --- |
| Latest | ✅ |

## Reporting a Vulnerability
Report security vulnerabilities via the project's issue tracker or contact the maintainers directly.

Include:
- Issue type
- Detailed description
- Steps to reproduce
- Potential impact

Please avoid public disclosure until a fix is provided.
